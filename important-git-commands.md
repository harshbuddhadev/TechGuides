# Important Git Commands

## Git clone
```bash
git clone <git repo .git link>
```

## Edit Repo Git Config
```bash
git config --edit
```
## Edit Global Git Config
```bash
git config --edit --global
```

## Git Clone specific branch

```bash
git clone <git repo .git link> --branch <branch-name>
```
